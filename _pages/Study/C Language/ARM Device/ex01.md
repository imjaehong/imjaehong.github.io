# 문제 설명
---
배열 a를 이용하여 x[2]를 30으로 만드는 코드를 만드시오

```c
#include <stdio.h> 

int x[4] = {1,2,3,4};

void main(void)
{
    int *a[4] = {x+3, x+2, x+1, x};
    
    printf("%d\n", x[2]);    

              = 30;   

    printf("%d\n", x[2]);
}
```

# 출력 예시
---

```
3
30
```

# 정답
---

```c
#include <stdio.h> 

int x[4] = {1,2,3,4};

void main(void)
{
	int *a[4] = {x+3, x+2, x+1, x};

	printf("%d\n", x[2]);

	//a[0][-1] = 30;
	//a[1][0] = 30;
	//a[2][1] = 30;
	a[3][2] = 30;
	

	printf("%d\n", x[2]);
}
```
