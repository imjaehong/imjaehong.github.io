# 문제 설명
---
반복문을 이용하여 다음 모양을 인쇄하는 함수를 설계하고 이름 main에서 호출하라.
단, printf 한 번에 '*' 한 번만 출력하고, for루프를 사용하여 구현하라.

# 출력 설명
---

```
'*' 문자를 출력 예시와 같이 출력한다.
```

# 출력 예시
---

```
*****
****
***
**
*
```

# 정답 코드
---

```c
#include <stdio.h>

void Draw_Start2(void)
{
	int line, star;

	for (line = 0; line < 5; line++)
	{
		for (star = 0; star < 5 - line; star++)
		{
			printf("*");
		}

		printf("\n");
	}
}

void main(void)
{
	Draw_Start2();
}
```

# 메모
---
> printf 내부의 **\n** 습관화 필요
