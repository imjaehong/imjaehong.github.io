# 문제 설명
---
10개의 정수(음수, 0, 양수 가능)가 공백으로 분리되어 num 배열에 저장된다.
10개의 값이 모두 다를 경우 YES, 하나라도 같은 값이 있으면 NO를 인쇄하는 코드를 구현하라.
아래 코드를 이용하되 주어진 코드는 변경하면 안되며 필요한 변수, 함수는 임의로 추가 가능하다. 

```c
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>

void main(void)
{
    int num[10];
    int i;

    for(i = 0; i < 10; i++)
    {
        scanf("%d", &num[i]);
    }

    // 코드 구현


}
```

# 입력 예시
---

```
1 9 2 8 7 3 66 55 44 100
```

# 출력 예시
---

```
YES
```

# 정답 코드
---

```c
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>

void main(void)
{
    int num[10];
    int i;

    for(i = 0; i < 10; i++)
    {
        scanf("%d", &num[i]);
    }

    int j, r = 0;

    for(i = 0; i < 10; i++)
    {
        for(j = i + 1; j < 10; j++)
        {
            if(num[i] == num[j])
            {
                printf("NO\n");
                return;
            }
        }
    }

    printf("YES\n");
}
```
