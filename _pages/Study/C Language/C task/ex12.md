# 정답 코드
---

```c
#include <stdio.h>

float compute_circle_area(float radious);

void main(void)
{
	float r;
	scanf("%f", &r);
	printf("%f\n", compute_circle_area(r));
}

float compute_circle_area(float radious)
{
	float pi = 3.14f;
	radious = radious * radious * pi;
	return radious;
}
```
