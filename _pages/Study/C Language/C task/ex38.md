# 정답 코드
---

```c
#include <stdio.h>

int main(void)
{
	for (int i = 1; i <= 6; i++)
	{
		for (int j = 1; j <= 6; j++)
		{
			for (int h = 1; h <= 6; h++)
			{
				printf("%d %d %d\n", i, j, h);
			}
		}
	}
	return 0;
}
```
