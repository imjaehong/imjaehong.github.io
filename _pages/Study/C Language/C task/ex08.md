# 정답 코드
---

```c
#include<stdio.h>

void func(void * p)
{
	int i;

	for(i=0; i<3; i++)
	{
		printf("%f\n", (*(double**)p)[i]);
	}
}

void main(void)
{
	double d[3] = {3.14, 5.125, -7.42};
	void *p = d;

	func(&p);
}
```
