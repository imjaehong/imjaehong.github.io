# 문제 설명
---
'A'부터 입력받은 글자(영문 알파벳 대문자)까지 알파벳을 인쇄하는 코드를 for 루프로 구현하라.

# 입력 예시
---

```
K
```

# 출력 예시
---

```
A
B
C
D
E
F
G
H
I
J
K
```

# 정답 코드
---

```c
#include <stdio.h>

void main(void)
{
	char i, c;

	scanf("%c", &c);

	for (i = 'A'; i <= c; i++)
	{
		printf("%c\n", i);
	}
}
```

# 메모
---
> printf 내부의 **\n** 습관화 필요
