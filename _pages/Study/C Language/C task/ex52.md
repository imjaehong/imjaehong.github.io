# 정답 코드
---

```c
#include <stdio.h>

char * Vending_Machine(int num)
{
	static char drink[4][10] = { "cola", "milk", "coffee", "wine" };

	return drink[num];
}

void main(void)
{
	int n;

	scanf("%d", &n);
	printf("%s\n", Vending_Machine(n));
}
```
