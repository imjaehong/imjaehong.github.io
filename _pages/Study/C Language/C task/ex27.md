# 문제 설명
---
*을 입력 받은 정수 n개 만큼 인쇄하는 코드를 for 루프로 구현하라.
단, 루프 1번에 한 개의 * 출력

# 입력 예시
---

```
8
```

# 출력 예시
---

```
7
8
9
10
11
12
13
14
```

# 정답 코드
---

```c
#include <stdio.h>

void main(void)
{
	int i;
	int n;
	
	scanf("%d", &n);

	for(i = 7; i < (n + 7); i++) 
	{
		printf("%d\n", i);
	}
}
```

# 메모
---
> printf 내부의 **\n** 습관화 필요
