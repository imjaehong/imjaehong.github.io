# 문제 설명
---
3부터 값을 증가시키며 3의 배수의 합을 구할 때 합이 1000이 넘지 않는 3의 배수를 인쇄하는 프로그램을 작성하시오. (즉, 3+6+9+12+...+N < 1000 을 만족하는 가장 큰 N을 인쇄)

# 출력 설정
---

```
3+6+9+12+...+N < 1000 을 만족하는 가장 큰 N을 인쇄
```

# 출력 예시
---

```
75
```

# 정답 코드
---

```c
# include <stdio.h>

int main()
{
    int sum = 0;
    int i;

    for (i=3; ;i+=3)
    {
        sum += i;
        if (sum > 1000) break;
    }
    printf("%d", i-3);
    return 0;
}
```
