# 문제 설명
---

```c
#include <stdio.h>
 
void func(void *p)
{
      printf("%s\n",          );
}
 
void main(void)
{
      char * p = "Willtek";
      func(&p);
}
```

# 정답 코드
---

```c
#include <stdio.h>

void func(void *p)
{
	printf("%s\n", (*(char**)p));
}

void main(void)
{
	char * p = "Willtek";
	func(&p);
}
```

# 메모
---
> printf 내부의 **\n** 습관화 필요
