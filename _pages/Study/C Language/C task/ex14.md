# 문제 설명
---
float 값을 넣으면 가장 가까운 정수 값을 찾아서 넘겨주는 함수를 설계하라
3.56는 4, 3.12는 3으로 반올림하여 가까운 정수를 리턴하되 음수 값은 입력하지 않는 것으로 한다

```c
#include <stdio.h>

int find_int(float value);

void main(void)
{
	float num;
	scanf("%f", &num);
	printf("%d\n", find_int(num));
}

int find_int(float value)
{

}
```

# 입력 예시
---

```
3.56
```

# 출력 예시
---

```
4
```

# 정답 코드
---

```c
#include <stdio.h>

int find_int(float value);

void main(void)
{
	float num;
	scanf("%f", &num);
	printf("%d\n", find_int(num));
}

int find_int(float value)
{
	float bias = 0.5f;
	value += bias;
	return (int)value;
}
```
