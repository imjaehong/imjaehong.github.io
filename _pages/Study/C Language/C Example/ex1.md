# 문제 설명
---
printf 함수를 이용하여 출력 예시와 같이 인쇄되도록 코드를 설계하라

```c
#include <stdio.h>

void main(void)
{
	
}
```

# 출력 예시
---

```
Hello C World!
Welcome!!
```

# 정답
---

```c
#include <stdio.h>

void main(void)
{
	printf("Hello C World!\n");
	printf("Welcome!!\n");
}
```

---
