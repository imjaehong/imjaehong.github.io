# 문제 설명
---
2단부터 9단 까지 구구단을 인쇄하는 프로그램을 설계하라
2단 부터 9단까지 아래 형식으로 인쇄하되 각 단은 구분기호를 추가한다
각 단의 구분은 printf("=======================\n"); 를 사용하라

```c
// 각 단의 구분은 아래 코드를 이용한다
// printf("=======================\n");

#include <stdio.h>

void main(void)
{
	// 코드 작성
}
```

# 정답 코드
---

```c
// 각 단의 구분은 아래 코드를 이용한다
// printf("=======================\n");

// 코드 작성
#include <stdio.h>
void main(void)
{
	int i, j;
	for (i = 2; i <= 9; i++)
	{
		printf("%d단\n", i);
		for (j = 1; j <= 9; j++)
		{
			printf("%d * %d = %d\n", i, j, i*j);
		}
		printf("=======================\n");
	}
}
```
