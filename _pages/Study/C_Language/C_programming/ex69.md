# 문제 설명
---
문자열 2개를 입력 받아 대소를 판단하는 코드를 설계하라
문자열의 대소 비교 규정 : 글자들을 순서대로 비교하면서 대소를 판단한다
대소 판단 예 : (1) KIM == KIM (2) Kim > KIM (3) abcd > abc (4) abx > abcdex
앞 문자열이 크면 “BIG”,  앞 문자열이 작으면 “SMALL”, 같으면 “SAME”을 출력한다
단, 입력되는 두 개의 문자열은 영문자 최대 10글자의 공백이 없는 문자열이다

```c
#include <stdio.h>

void main(void)
{
	char a[11];
	char b[11];

	scanf(" %s", a);
	scanf(" %s", b);

	// 코드 구현
}
```

# 입력 예시
---

```
KIM
KIM
```

# 출력 예시
---

```
SAME
```

# 정답 코드
---

```c
#include <stdio.h>

void main(void)
{
	char a[11];
	char b[11];
	int i = 0;

	scanf(" %s", a);
	scanf(" %s", b);

	// 코드 구현
	while (a[i] != 0 || b[i] != 0)
	{
		if (a[i] > b[i])
		{
			printf("BIG\n");
			return;
		}
		if (a[i] < b[i])
		{
			printf("SMALL\n");
			return;
		}
		i++;
	}
	printf("SAME\n");
}
```
