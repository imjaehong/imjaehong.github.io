---

리눅스~~
```c
#include <stdio.h>

void main(void)
{
	
}
```

# 출력 예시
---

```

Welcome!!
```

# 정답 코드
---

```c
#include <stdio.h>

void main(void)
{
	printf("Hello C World!\n");
	printf("Welcome!!\n");
}
```
